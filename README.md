# ➤ Single

Um jogo inspirado em Wordle, onde você precisa adivinhar o artista.

## Como iniciar o app localmente

1. Clonar o projeto

```bash
  git clone https://github.com/M4ricu/A3-Frontend/
```

2. Ir para a pasta do projeto

```bash
  cd A3-Frontend
```


3. Instalar dependências
```bash
  npm install
```

4. Iniciar servidor
```bash
  npm run dev
```
